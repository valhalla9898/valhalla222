## Security Flow

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ                    Security Measures                         в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ

1. Password Storage
   в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
   User Password вҶ’ SHA-256 Hash вҶ’ Store in Database
   
   Example:
   "admin123" вҶ’ hash вҶ’ "240be518fabd2724ddb6f04eeb1da5967448d7e831c08c8fa822809f74c720a9"

2. Authentication
   в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
   Login Attempt вҶ’ Retrieve User вҶ’ Hash Input Password вҶ’ Compare Hashes
   
   If Match: Create Session
   If No Match: Reject Login

3. Authorization
   в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
   Page Request вҶ’ Check Session вҶ’ Check Role вҶ’ Allow/Deny
   
   Admin Pages: Require role = 'admin'
   User Pages: Require role = 'user' OR 'admin'

4. Session Management
   в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
   Login Success вҶ’ Store in st.session_state:
   {
     'authenticated': True,
     'current_user': {
       'id': 1,
       'username': 'admin',
       'role': 'admin',
       'full_name': 'System Administrator',
       'email': 'admin@example.com'
     }
   }
   
   Logout вҶ’ Clear session_state вҶ’ Redirect to Login
```
